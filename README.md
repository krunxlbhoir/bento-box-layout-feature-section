# Bento Box Layout Feature Section

A modern bento box style feature section built using **HTML and CSS Grid**.  
This project focuses on creating a clean, responsive, and accessible layout
commonly used in modern landing pages and product websites.

## Demo
https://krunxlbhoir.github.io/bento-box-layout-feature-section/

## How to run locally
Open `index.html` in any modern web browser.

## Features
- CSS Grid based bento box layout
- 5 responsive feature cards
- Clean typography and spacing
- Fully responsive across desktop, tablet, and mobile
- Semantic HTML for better accessibility and SEO
- No external libraries or frameworks

## Tech Stack
- HTML5
- CSS3 (Grid & Flexbox)

## Notes
- Uses CSS Grid for complex layout structure
- Designed without images for better performance
- Easily customizable for real projects or portfolios
